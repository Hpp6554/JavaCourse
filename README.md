## 配置Eclipse和开发环境
0. JDK下载和环境变量设置（略）；
1. 从Eclipse官网下载最新版本的[编辑器](http://www.eclipse.org/)，裸奔的编辑器已经带Maven，不需要另外安装；
2. 修改编辑器字体Consolas，一定要设置为中文的字体号，否则windows下中文会变得异常；
3. 设置Maven仓库，用阿里云的仓库，可以大大加快依赖包的下载速度；
